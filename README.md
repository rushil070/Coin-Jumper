# Coin Jump
A single button arcade style game where the player's job is to attain 20 coins by jumping at the correct time.

 - The goal of the game is for the user to earn 20 coins by jumping at the right time when the coin appears.
 - The player can jump by clicking Space.
 - Coins move continuously to the left, while the player, who is stationary, must jump to collect them.
 - Four Buttons are present
      - Start Game: Begins the game in normal mode
      - Hard Mode: Quickens the speed of the coin
      - Easy Mode: Slows the speed of the coin
      - Reset Game: Resets the coin in the correct position
  - A welcome pop up is shown which gives the instructions of the game.
- An ending pop up shows up once the game is finished congratulating the user.
